# RokuyouOfHoliday_gas
結婚式が挙げられることの多い土日祝日における六曜をGoogleカレンダーに表示するGoogleAppsScriptです。  
ブライダルバイト民がシフト計画を立てる際の参考にでもなればいいなと思っています。

既存コードの改変・転用をかなり雑に処理してしまったので、後ほど改善する予定です。

googleカレンダーで2018年度版を公開しています。
